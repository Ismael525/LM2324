Tablas

Una tabla permite la distribucion de la informacion organizada por filas(rows) y columnas(cols). Se utiliza la etiqueta html de nombre table. Ejemplo:

Elemento 1 Elemento 2
Elemento 3 Elemento 4

Problemas...

Vamos a experimentar problemas cuando el numero de columnas no sea el mismo en todas las filas. LO siguiente esta mal... Ejemplo:

Elemento 1
Elemento 3Elemento 4

Soluciones

Si deseamos que una celda ocupe mas de una columna o fila entonces utilizaremos las propiedades colspan o rowspan. Ejemplo:

Elemento 1
Elemento 3 Elemento 4
